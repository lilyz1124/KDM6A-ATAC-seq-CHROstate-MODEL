# Kdm6a Heart Pipeline

This repository analyzes mouse heart ATAC-seq and chromatin-state data, integrates Kdm6a perturbation expression data, and tests sex-specific effects.

Public resources motivating the design include ENCODE mouse heart ATAC annotation, mouse chromatin-state resources, and a KDM6A sex-biased perturbation study showing loss of chromatin accessibility at dysregulated promoters.
